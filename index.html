Get Started

To start collecting performance metrics, follow these steps.

1 Install Speed Insights

เนื่องจากเราใช้ HTML ธรรมดา ให้ใช้ Script นี้แทนการใช้ npm:

<script defer src="/_vercel/insights/script.js"></script>
2 Check Web Vitals

ระบบจะวัดค่า LCP, FID, และ CLS ของหน้าเว็บคุณโดยอัตโนมัติ

import { SpeedInsights } from "@vercel/speed-insights/next" // สำหรับ Next.js
3 Deploy & Visit your Site

Push งานขึ้น GitHub แล้วไปที่ URL ของคุณเพื่อเริ่มเก็บข้อมูลได้เลย!
